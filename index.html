Transit Inequity in the San Francisco Bay Area

Visualizing public transit access gaps and demographic impacts

Overview

This project investigates spatial inequities in public transit access across the Bay Area. Using data from the National Transit Database and U.S. Census, we created dashboards to visualize areas lacking reliable transit and explore how these gaps correlate with socioeconomic characteristics such as income, race, and vehicle ownership.

Dashboard 1: No Transit Access Zones

This dashboard highlights areas that fall outside a ¼-mile walkable buffer of any public transit stop (bus or train). These zones represent "true transit deserts" — communities with no nearby transit infrastructure.

Dashboard 2: Low-Frequency Transit Areas

This dashboard displays areas where transit stops exist but service frequency is below a usable threshold (e.g., fewer than 3 trips per hour). These areas reveal the hidden inaccessibility masked by apparent coverage.
